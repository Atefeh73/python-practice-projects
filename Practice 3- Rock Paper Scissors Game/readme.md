**Rock Paper Scissors Game**

A Python console-based Rock, Paper, Scissors game featuring both:
  - Single-player
  - Two-player

This project was created as part of my Python learning journey to practice programming fundamental, game logic, functions, loops, moudles, and user interaction.

__________________________________________________________________________________________________________________________________________________________________

**Game Modes**
  1. Single Player Mode:
     - Computer generates random choices using Python's random module.
     - Best-of-3 rounds
     - Score tracking system
  2. Two Players Mode:
     - Hidden player input using Python's getPass modul
     - Fair turn-based gameplay
     - Best-of-3 rounds
     - Real-time score updates

  **Game Rules**
  - Rock(r) Vrs Scissors(s)
  - Scissor(s) Vrs Paper(p)
  - Paper(p) Vrs Rock (r)
  The player with the highest core after 3 rounds wins the game!

**Hidden Input System**
 Two-player mode uses Python’s getpass module so player choices are hidden while typing.

 This prevents players from seeing each other's moves during gameplay.
